# 🌐 Desenvolvimento Web - IFSP Pirituba (Semana 03)

Repositório de prática de HTML5 focado em semântica, textos, listas, imagens e hiperlinks.

## 📖 Guia Rápido de Tags
- `<html>`: Raiz do documento.
- `<head>`: Metadados e título.
- `<body>`: Conteúdo visível.
- `<a>`: Hiperlinks (internos e externos).
- `<img>`: Inserção de mídia visual.
- `<h1>`-`<h6>`: Hierarquia de títulos.
- `<div>`: Agrupamento de elementos.
- `<footer>`: Rodapé institucional.

## 🧠 Conceitos Aplicados
- **Navegação:** Uso de âncoras (`#top`).
- **Targeting:** Abertura de links em novas abas.
- **Acessibilidade:** Descrições `alt` em imagens.
